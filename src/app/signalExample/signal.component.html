Signal inputs replace @Input
The new signal input feature replaces the old @Input logic and greatly reduces the logic needed to detect changes to passed in values. Previously you would need to wire in logic to detect when a the input value changed; now the input parameters are signals, so you can use computed values or effects that automatically detect changes.
