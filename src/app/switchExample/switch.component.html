@switch replaced ngSwitch for conditionally displaying content
The switch directive is a replacement for the ngSwitch / ngSwitchCase extension. It basically does the same things but it's much more performant and doesn't cause as many repaints when processing array data.
https://blog.angular-university.io/angular-switch/
switch in action:
@for(chick of chickens; track chick.breedID) {
{{ chick.breedName }}
@switch (chick.weight) { @case(3) { small } @case(4) { average } @case(5) { large } @case(6) { giant } @default() { unknown } }
}
