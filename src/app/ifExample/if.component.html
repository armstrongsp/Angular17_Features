@if replaced ngIf for conditional display
The if directive is a replacement for the ngIf extension. It works the same as ngIf but it more performant.
https://blog.angular-university.io/angular-if/
an if / else statement
@if (someControlValue == "A") {
The thing is equal to A
} @else if (someControlValue == "B") {
The thing is equal to B
} @else {
The thing is not A or B
}
