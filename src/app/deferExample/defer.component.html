@defer allows you to dynamically load parts of a page based on an event
The components referenced inside of the defer clause will be lazy loaded when the triggering event occurs. There are some built in events that can be used, or you can define your own custom events.
https://blog.angular-university.io/angular-defer/
defer using the built in 'on hover' event:
@defer (on hover(loadFor)) { } @placeholder {
hover here to load
} @loading (minimum 4s) {
....the component is loading
}
Example code
defer using a custom trigger:
@defer (when showDeferNow) { } @placeholder {
click here to show it
}
